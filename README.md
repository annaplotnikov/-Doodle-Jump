🐸 Doodle Jump – Консольный клон 
Классическая аркада «Doodle Jump» прямо в вашем терминале!
Управляйте забавным персонажем, прыгайте по платформам, собирайте монетки и устанавливайте рекорды.
Реализована на семи языках программирования с единым геймплеем, цветной анимацией, системой очков и рекордов, движущимися платформами и бонусами.

🚀 Возможности
Игровой процесс – персонаж автоматически прыгает при касании платформы. Игрок управляет горизонтальным движением (← →) и стремится подняться как можно выше.

Платформы – генерируются случайным образом, некоторые движутся, другие разрушаются после прыжка.

Бонусы – монетки (увеличивают счёт) и ускорители (временное увеличение скорости прыжка).

Счёт и рекорд – количество пройденных платформ и собранных монет сохраняется в локальный файл.

Сложность – скорость подъёма увеличивается по мере набора очков.

Цветная графика – персонаж, платформы и фон окрашены для лучшей визуализации.

Управление – стрелки влево/вправо для перемещения, пробел для ускоренного падения, Q для выхода, R для перезапуска.

Кроссплатформенность – работает в любом терминале с поддержкой ANSI (Linux, macOS, Windows с WSL или ConEmu).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --speed <N>	Базовая скорость игры (задержка в мс, по умолчанию 50)
-h, --help	Справка
Управление
Клавиша	Действие
← / A	Движение влево
→ / D	Движение вправо
Пробел	Ускоренное падение (вниз)
R	Перезапуск после окончания игры
Q	Выход из игры
🛠 Установка и запуск
Python

python doodle.py [опции]
C++
bash

Требуется библиотека ncurses (установите через пакетный менеджер).

Ruby

Java

🧠 Логика игры
Инициализация – персонаж находится в нижней части экрана, платформы генерируются снизу вверх.

Движение – персонаж автоматически движется вверх при прыжке, игрок управляет горизонтальным смещением.

Физика – гравитация действует постоянно; при касании платформы персонаж подпрыгивает.

Платформы – появляются случайным образом; некоторые движутся влево-вправо, другие исчезают после прыжка.

Столкновения – если персонаж падает ниже экрана, игра заканчивается.

Счёт – увеличивается за каждую пройденную платформу и собранную монетку.

Рекорд – сохраняется в файл и отображается на экране.

Бонусы – монетки ( +10 очков) и ускорители (временное увеличение скорости прыжка).

✨ Дополнительные фичи
Анимированный персонаж – меняет цвет или форму при прыжке.

Динамическая сложность – скорость подъёма увеличивается с каждым десятком очков.

Звуковые эффекты – при прыжке и сборе монет (ASCII bell).

Пауза – нажмите P для приостановки игры.

Автосохранение – рекорд обновляется автоматически.
